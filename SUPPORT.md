# Sunriser – Alarm Support

If you need help, want to report a bug, or have feedback, contact:
- **vyas.chitti@icloud.com**

## Common Fixes

### Sunrise time or weather not loading
- Confirm **Location Services** are enabled for the app:
  - iOS Settings → Privacy & Security → Location Services → **Sunriser – Alarm**
- If you don’t want to share location, choose a manual location inside the app (if available).

### Sunrise alert didn’t fire
- Confirm **Notifications** are enabled:
  - iOS Settings → Notifications → **Sunriser – Alarm** → Allow Notifications
- Make sure **Focus / Do Not Disturb** isn’t silencing notifications.
- Toggle the sunrise alarm **off and back on** to reschedule it.

## App Info
